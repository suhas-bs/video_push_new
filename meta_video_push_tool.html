⚠️ Prevent ID precision loss when exporting from Google Sheets
Before downloading as CSV, select the campaign_id, adset_id, and product_set_id columns → Format → Number → Plain text. This stops Sheets from rounding 18-digit numbers.

Status legend

✅ Success — Ad created (PAUSED in Meta) ❌ Failed — API returned an error (see error_message) ⚠️ Ineligible — Post not eligible for partnership ad ⏭️ Skipped — Row had missing or invalid data
💡

Notes

Token safetyYour access token is used only for the session and is never stored, logged, or shared.
Account IDsThe sidebar has Flipkart Minutes defaults pre-filled. Any team member can edit them without touching the code.
Ad statusAll ads are created in PAUSED state. Review in Meta Ads Manager before activating.
Re-runsFilter failed rows from the results CSV, fix the issue, re-export, and re-upload — only failed rows will be retried.
Verify configExpand "🔍 Verify Account Config" at the top of the tool before running to confirm your Meta account linkage is correct.
First time using the tool? Expand the Verify Account Config section, paste your token, and click Verify — it will show you which Instagram account is linked to your Facebook page and confirm everything is set up correctly before you push.
